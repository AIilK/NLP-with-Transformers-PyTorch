**Educational repository** for learning Natural Language Processing — from classical text preprocessing techniques to modern **Transformer** models using PyTorch and Hugging Face.

---

## 📌 About This Repository

This repo documents my journey in NLP, covering both traditional methods and state-of-the-art Transformer-based approaches.

### What You'll Find:

- **Classical NLP Techniques**
  - Text preprocessing (stopwords, stemming, lemmatization, unicode cleaning)
  - Token alignment and basic text processing
  - Kaggle-style workflows

- **Modern Transformer Models**
  - Sentiment Analysis using Hugging Face Transformers
  - Sentiment Analysis with Flair
  - Ready-to-use PyTorch + Transformers pipelines

## 🚀 Quick Start

### Installation

```powershell
# Clone the repo
git clone https://github.com/AIilK/NLP-with-Transformers-PyTorch.git
cd NLP-with-Transformers-PyTorch

# Install dependencies
pip install -r requirements.txt
```

### `requirements.txt` (پیشنهادی)

```txt
torch
transformers
datasets
flair
pandas
numpy
scikit-learn
matplotlib
tqdm
```
